# MiniPlay Hub

A responsive, multi-page React mini-game hub built with Vite, Tailwind CSS, and React Router.

## What is included

- Home, Games, Single Game, About, Contact, and 404 routes
- Filterable game catalog with reusable cards and shared layout
- Five client-side games:
  - Snake
  - Tic-Tac-Toe
  - Memory Match
  - Rock Paper Scissors
  - Number Guess
- Keyboard, mouse, and touch-friendly controls
- Client-side contact form validation with confirmation state

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Project structure

```text
src/
  components/
  data/
  games/
  pages/
```

Future features such as leaderboards, sound effects, daily challenges, login, score saving, and a PWA wrapper can be added without changing the current game component model.
